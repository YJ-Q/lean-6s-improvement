# Lean 6S Skill 测试资料

> 本目录记录合成用户测试，用于在真人招募前发现输入、追问和规则问题。它不等同于真实用户研究，不能证明满意度、效率提升、业务收益或真实世界准确率。

## 目录指南

- `personas.json`：5 个冻结的模拟人物卡。
- `scenarios.json`：3 类现场任务及只可按追问揭示的隐藏事实。
- `rubric.json`：8 个 0–2 分的形成性评分维度。
- `protocol.md`：会话生成、条件隔离和复核流程。
- `sessions/`：30 份合成会话，包括普通 AI 和 Skill 两种条件。
- `results.json` 与 `report.md`：可重现聚合结果与局限说明。

会话和评分都是 AI 中介的形成性材料，不应当作真人证言、统计证据或业务成效。
