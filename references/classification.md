# 6S 分类规则

使用可观察的现场逻辑，而非仅凭关键词。确定一个主要类别和可选的次要类别。

## 整理 / Seiri / Sort

当主要失效是多余物品、过期物料、过量在制品、废料、过期消耗品或保管状态不清时使用。

典型证据：

- 不用的夹具、旧模具、过期化学品、废品与良品混放。
- 过量库存堵塞作业区域。
- 没有使用人、没有订单、没有下工序的物品。

可执行的对策：

- 红牌标识物品，隔离区域，明确处置责任人和期限。
- 设定保留/废弃标准和最大数量。
- 从标准区域布局中移除过时物品。

## 整顿 / Seiton / Set in Order

当必需的物品存在但缺乏固定位置、标识、数量标准、取用逻辑或归位纪律时使用。

典型证据：

- 工具放在机器上、地上、货架边缘或随意推车上。
- 物料存在但缺少标识、最大/最小量、先进先出或位置编码。
- 操作人员需要翻找、多走路或借用非正式的替代品。

可执行的对策：

- 定义固定位置、标识、形迹板、线标、最大/最小数量和归位责任人。
- 增加异常数量或错误放置的目视管理。
- 更新使用点布局。

## 清扫 / Seiso / Shine

当主要失效是污染、脏污、泄漏、切屑、灰尘、油污、废弃物堆积或清洁源头控制时使用。

典型证据：

- 油/水泄漏、金属切屑、灰尘堆积、机器周围垃圾。
- 只在检查前才清洁。
- 污染源头未被修复。

可执行的对策：

- 立即清洁，识别源头，修复泄漏或增加收集方法。
- 定义清洁点、方法、工具、频率、责任人和验收照片。

## 清洁 / Seiketsu / Standardize

当现场缺乏稳定的标准来防止整理、整顿和清扫的退步时使用。

典型证据：

- 不同班组使用不同的放置规则。
- 点检表存在但不包含此问题。
- 缺少标准照片、审核点、责任人或升级规则。

可执行的对策：

- 创建或修订标准照片、点检表、标识模板、审核路线和异常升级规则。
- 按班组培训并通过反复检查验证。

## 素养 / Shitsuke / Sustain

当已知的标准存在但人员反复不遵守、管理者容忍、或仅在审核期间行为改变时使用。

典型证据：

- "大家都知道但还是这么做。"
- 培训后反复不合规。
- 在客户或领导检查前临时整改。

可执行的对策：

- 确认标准清晰度，重新培训相关人员，增加分层审核、反馈机制和问责制。
- 在指责操作人员之前，先检查标准是否切合实际、工具是否可用。

## 安全 / Safety

当主要风险是伤害、火灾、堵塞逃生/消防设备、机器防护、电气风险、化学品暴露、叉车/行人冲突、物体坠落或不安全通道时使用。

典型证据：

- 堵塞灭火器、紧急出口、配电柜或通道。
- 缺少防护罩、暴露的锋利边缘、油污滑倒风险、堆码货物可能倒塌。
- 在必需区域缺少 PPE。

可执行的对策：

- 立即控制，隔离危险，恢复安全防护，并要求主管验证。
- 通过布局、标识、物理屏障、上锁挂牌、点检表或升级机制预防复发。

## 主分类和副分类原则

先判断主要失效，再判断风险后果。主分类表示当前最需要管理者控制的失效类型；副分类表示明显相关、但不是第一控制点的 6S 类别。

- 如果存在即时人身伤害、消防、逃生、电气、机械防护、叉车人车冲突、坠落、滑倒等风险，通常将安全作为主分类，其他类别作为副分类。
- 如果没有即时安全风险，优先按"问题为什么会发生"而不是"问题造成什么不舒服"分类。
- 如果问题来自物品是否该存在，优先整理；如果物品需要但位置和取放规则混乱，优先整顿。
- 如果问题来自当前脏污、泄漏、切屑、积尘，优先清扫；如果问题来自缺少保持标准、频率、责任人或审核机制，优先清洁。
- 如果标准清楚、资源可用、责任明确，但人员或班组反复不执行，才将素养作为主分类。
- 副分类不要超过两个。只有当副分类会改变对策时才列出，例如"安全 + 整顿"会要求先恢复通道，再建立定置和防堵机制。

### 副分类判定规则

副分类用于提醒治理时必须同时覆盖的次要失效。不要把所有相关词都写成副分类；只有当副分类会改变对策、验收证据或复查重点时才列出。

必须写副分类：

- 主分类不是安全，但存在火灾、绊倒、滑倒、坠落、叉车碰撞、消防遮挡、电气或化学品暴露等风险时，加安全。
- 主分类是安全，但根因来自物品放置、管线收纳、通道定置、数量管控或取放路径混乱时，加整顿。
- 主分类是安全，但根因来自不需要物品、废弃物、过期物或呆滞物未清理时，加整理。
- 主分类是安全，但直接危险来自油污、积水、粉尘、料粒、泄漏或污染物时，加清扫。
- 主分类是清扫，且改善必须建立清扫频率、责任人、点检表、标准照片或复查机制时，加清洁。
- 主分类是整顿，且现场同时存在低频、闲置、报废、不需要或超量物品需要从主区域清理时，加整理。
- 主分类是整理，且保留物品还需要定置、标签、容量、先进先出或货架编码时，加整顿。
- 主分类是素养，且不执行行为带来人身伤害、消防、电气、化学品或叉车风险时，加安全。

不要写副分类：

- 只是描述中出现相关词，但不改变整改动作、验收证据或复查重点。
- 主分类已经覆盖了主要对策，副分类只会重复同一件事。
- 只有轻微、间接、可当班消除的影响，且无需形成单独控制措施。
- 证据不足以证明另一个类别真实存在，例如只说"可能有风险"但没有风险场景、位置或对象。

可选写副分类：

- 主分类是整顿，现场有少量当前脏污，但整改重点不是污染源或清扫制度时，清扫可作为副分类，也可只在整改建议中说明。
- 主分类是安全，现场同时有整理和整顿失效时，只保留更能解释根因、且会改变对策的一项副分类。
- 主分类是清洁，现场有当前脏污或位置混乱时，只有需要同步清理或重新定置，才加清扫或整顿。
- 主分类是整顿或清洁，且存在标准已明确但反复不执行的证据时，才加素养；首次系统改善、资源不足、布局不合理不加素养。

## 易混淆边界

### 整理 vs 整顿

判为整理：

- 物品是否需要保留不清楚，例如无订单、无使用人、无下工序、无领用计划。
- 物品过期、报废、呆滞、超量，或良品、废品、待判品混在一起导致状态不清。
- 仓库或现场面积被不需要物品占用，核心问题是"该不该存在"。

判为整顿：

- 物品是需要的，但没有固定位置、位置编码、标签、最大/最小量、先进先出规则。
- 工具、备件、劳保用品、办公用品等需要频繁取用，但现场需要翻找、借用、绕路或凭经验寻找。
- 物品有归属区域，但放置边界、容器、货架、形迹板、看板信息不清。

容易误判：

- "物料混放"不一定是整理。如果混放的是需要物料，主要问题是缺少位置、标识、批次或数量规则，优先判整顿。
- "库存多"不一定是整理。如果超过经济批量或最高库存标准，且保管状态清楚，整理为主；如果没有最高/最低量和可视化标准，整顿或清洁也可能作为副分类。

### 清扫 vs 清洁

判为清扫：

- 当前现场存在油污、积水、粉尘、切屑、垃圾、泄漏、污染源或清洁死角。
- 需要立刻清除污染，并确认污染来源。
- 设备周边、维修室、仓库角落等区域已经影响作业、质量、设备或安全。

判为清洁：

- 当前问题反映出缺少清扫标准、标准照片、点检频率、责任人、审核路线或异常升级规则。
- 不同班组、不同区域采用不同做法，且原因是缺少统一标准、培训、点检或审核机制，说明整理、整顿、清扫无法稳定保持。
- 点检表存在但没有覆盖该异常，或看板/可视化信息无法让管理者及时发现问题。
- 如果标准和管理机制已存在，但不同班组仍做法不一，应考虑素养作为主分类或副分类。

容易误判：

- "现场很脏"首先通常是清扫；如果清扫后反复变脏，且没有频率、工具、责任人和复查机制，则清洁作为主分类或副分类。
- "做了可视化看板"不自动等于清洁。只有看板承载标准、责任、频率、状态和异常反馈时，才支持清洁类对策。
- "叉车卫生差、定位线脏污、站台裂缝积料"等当前可见污染，先判清扫；只有当规则、频率、责任和复查机制缺失是主要失效时，才把清洁作为主分类。

### 整顿 vs 素养

判为整顿：

- 没有明确位置、标签、线标、容器、货架编码、形迹板或取放路径。
- 标准有但不容易执行，例如距离过远、容量不足、容器不匹配、拿取不顺手。
- 员工只能靠经验或临时变通才能完成取放。

判为素养：

- 标准清楚、位置合理、工具可用、责任明确，但仍反复不归位或不按标准执行。
- 培训、宣贯或审核后仍重复出现同类不合规。
- 现场人员或管理者把异常当作正常状态，例如"大家都知道但一直这样"。

容易误判：

- 不要把没有标准的问题直接判为素养。先检查标准、工具、空间、流程是否可执行。
- 首次系统性改善、布局重新规划、工具架重排、区域重新定置，通常先判整顿或整理，不因"正在推进""需要坚持"直接判素养。
- 如果只靠提醒才能维持，而没有分层审核、问题追踪和责任反馈，素养可作为副分类，但清洁或整顿仍可能是主分类。

### 清洁 vs 素养

判为清洁：

- 缺少统一标准、责任制度、检查频率、巡检表、异常反馈或问题追踪机制。
- 管理者标准作业、班组点检或看板状态没有把异常纳入日常管理。

判为素养：

- 标准和管理机制已经存在，且员工知道要求，但日常执行持续偏离。
- 只有在客户、领导或审核前才临时恢复标准状态。

### 安全 vs 其他 6S

判为安全主分类：

- 异常直接影响消防设施、逃生通道、配电柜、机械防护、化学品、叉车路线、登高、滑倒或坠落风险。
- 风险需要当班立即隔离、停止、恢复或主管确认。
- 异常阻断消防、逃生、配电、主物流通道或人员必经路径，或存在物体坠落、绊倒、叉车碾压、火灾等即时伤害风险。
- 即使根因是整理、整顿或清扫，只要当前后果可能造成伤害或重大事故，安全优先。

判为其他 6S 主分类、安全副分类：

- 风险较低，主要问题仍是物品状态、位置、清洁保持或标准化不足。
- 安全影响是间接后果，不需要立即停用、隔离或恢复安全功能。
- 安全风险可通过当班隔离或警示临时控制，且未阻断消防、逃生、配电、主物流或人员必经路径时，可保留原 6S 失效为主分类，将安全作为副分类。
- 废袋子、废旧物资、桶装水、吨袋等主要问题是"该不该留、留多少、放哪里、如何定置"时，优先按整理或整顿主分类；若存在易燃、占道、倾倒、绊倒等风险，补安全副分类。

示例：

- 托盘堵住灭火器：安全为主，整顿为副。
- 过期化学品未隔离且标识不清：安全为主，整理为副。
- 工具放在地上但不在通道、无绊倒风险：整顿为主。
- 油污在通道造成滑倒风险：安全为主，清扫为副。
- 废旧夹具占用非主通道但核心失效是不需要物品未清理：整理为主，安全为副。
- 设备底座下方小范围泄漏且可立即隔离，核心失效是泄漏源和污染清理：清扫为主，安全为副。
- 废袋子靠近热源、电气柜或消防设施：安全为主，整理为副；仅一般堆积且无即时风险时，整理为主，安全为副。
- 封口机气管、电缆、软管横跨人员或叉车路径且有绊倒、碾压、掉落风险：安全为主，整顿为副。

## 现场例子

整理：

- 仓库存在长期无人领用的备件、旧夹具或废旧物资，占用货架和通道。
- 劳保用品、办公用品库存明显超出需求，且没有最高库存或处置规则。
- 良品、废品、待检品放在同一区域，状态不清。

整顿：

- 备件库、劳保用品库、维修室没有位置编码、定置线或标签，取用靠熟人经验。
- 工具使用后放在设备、推车或货架边缘，没有归位槽和责任规则。
- 物料有固定区域但没有最大/最小量、先进先出、批次或看板信息。

清扫：

- 设备周边有油污、积水、粉尘、切屑或包装废弃物。
- 清洁后很快再次出现泄漏，污染源没有处理。
- 维修室、仓库角落或通道存在长期积尘和废弃物。

清洁：

- 点检表没有覆盖关键 6S 异常，管理者巡检无法稳定发现问题。
- 看板只展示口号或结果，没有责任人、标准状态、检查频率和异常反馈。
- 不同班组对同一物品使用不同放置规则。

素养：

- 标准、标签、线标都存在，但员工反复不按位置归还工具。
- 检查前现场恢复，检查后又回到原状态。
- 培训和提醒后仍重复出现同类不合规，且管理者默认接受。

安全：

- 消防栓、灭火器、配电柜、紧急出口或主通道被物品遮挡。
- 叉车路线和行人路线无隔离，存在人车混行风险。
- 设备防护罩缺失、化学品暴露、油污导致滑倒或物体堆码可能倒塌。

## 证据不足时的追问

分类不稳定时，优先追问能改变分类的证据：

- 这个物品是否还需要使用？有没有订单、使用人、下工序或保留期限？
- 当前是否有固定位置、标签、数量标准、点检表或责任人？
- 这是否已经造成安全、质量、设备、交付或环境风险？
- 问题是一次出现，还是清理/整改后又复发？
- 不同班组或区域是否采用不同做法？

## 分类输出格式

返回：

- 主要类别。
- 如有需要，次要类别。
- 为何选择此类别。
- 哪些证据会改变分类结果。
